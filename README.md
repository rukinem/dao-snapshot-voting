# 🏛️ dao-snapshot-voting

Off-chain snapshot voting integration with on-chain execution bridge

## Features

- Off-chain voting
- On-chain execution
- Signature verification
- Proposal sync
- Gas efficient

## Tech Stack

- **Smart Contracts:** Solidity ^0.8.20
- **Framework:** Hardhat
- **Frontend:** Next.js + TypeScript
- **Web3:** ethers.js / wagmi

## Getting Started

```bash
npm install
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network localhost
```
